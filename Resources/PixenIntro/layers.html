Layers

Pixen is one of only a few pixel editors to support editing by layers. If you've never used them before, you're missing out -- layers allow for tremendous flexibility, even in pixel art.

For instance, you can draw a character, then make a new layer for his clothing and draw it all in the new layer. Changes you make only affect the new layer, so if you make a mistake and erase a pixel, you won't be erasing the colors in the original layer.

A list of layers is displayed in the left pane of each document window.
